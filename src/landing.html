API — POST /ask
POST
/ask
Send a natural language question. Returns a grounded answer from the database.
Request
curl -X POST https://provue-tara-production.up.railway.app/ask \
  -H "Content-Type: application/json" \
  -d '{"question": "How much did I spend in total?"}'
Response
{
  "answer": "Your total net spend across all transactions is ₹3,54,786.19, which includes gross spend of ₹3,65,975.44 and refunds of -₹11,189.25."
}

/health
